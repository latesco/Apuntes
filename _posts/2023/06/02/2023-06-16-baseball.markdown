#### La Funci贸n.

La siguiente funci贸n, cambiar谩 el tipo de datos de algunas columnas en la tabla <i>Master.csv</i>.

```python
def para_sql(ruta):
    df = pd.read_csv(ruta)
    if os.path.split(ruta)[1] == 'Master.csv':
        df['debut'] = pd.to_datetime(df.debut, format='%Y-%m-%d')
        df['finalGame'] = pd.to_datetime(df.finalGame, format='%Y-%m-%d')
        for col in cols:
            df[col] = df[col].fillna(-1)
            df[col] = df[col].astype(int)
            df[col] = df[col].astype(str)
            df[col] = df[col].replace('-1', np.nan)
    return df
```
